Install Flow

설치 절차

01
파일 다운로드

메인 번들과 필요한 개별 파일을 다운로드합니다.

02
환경설정 준비

`.env`를 작성하고 compose 파일과 이미지 태그를 확인합니다.

03
이미지 준비

온라인은 pull, 오프라인은 `docker load`로 이미지를 준비합니다.

04
실행 및 확인

`docker compose up -d` 후 마이그레이션과 health 체크를 수행합니다.
