Install Flow

설치 절차

01
번들 다운로드

배포 번들을 받고 압축을 해제합니다. 번들 안에는 tar 이미지와 가이드가 포함됩니다.

02
환경설정 준비

`.env.release.example`를 `.env`로 복사한 뒤 DB 비밀번호와 LLM 주소를 확인합니다.

03
이미지 등록

`docker load -i safeagent-api-v0.2.0.tar`와 `docker load -i safeagent-portal-v0.2.0.tar`를 실행합니다.

04
실행 및 확인

`docker compose up -d`와 마이그레이션 실행 후 `3000`, `8000/docs`에서 정상 구동을 확인합니다.
